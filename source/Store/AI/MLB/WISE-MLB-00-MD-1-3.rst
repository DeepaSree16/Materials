WISE ML 202 Documentation
===========================

This document captures the design of WISE ML 202 Module Coverage.

Learning Outcomes
-----------------

At the end of the module the participants will be able to

    * understand machine learning concepts and range of problems that can be handled by machine learning  
    * apply ML and AI techniques in applications which involve perception, reasoning and learning

Schedule
---------

This is a 36-hour intensive training module. The module is categorized into twelve units; Each unit is divided into three sessions; One hour each.

Coverage Overview
------------------

* Classical ML Techniques
* ML Engineering and Practices 
* Unsupervisied Learning
