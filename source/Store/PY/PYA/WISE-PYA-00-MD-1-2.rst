WISE Python 100 Module documentation
====================================

This document captures the design of WISE Python 100 Module Coverage.

Learning Outcomes
-----------------

At the end of the module the participants will be able to

    * understand fundamental programming concepts
    * read and understand what a piece of code is doing
    * implement basic programming structures such as if statements, loops and functions

Schedule
--------

This is a 15-hour intensive training module. The module is categorized intofive units; Each unit is divided into three sessions; One hour each

Coverage Overview
-----------------

    * Introduction to Python
    * Tool Chain
    * Hello world Program
    * Building blocks of Python
    * Conditional statements
    * While loop
    * Functions
    * Lists and for loop
