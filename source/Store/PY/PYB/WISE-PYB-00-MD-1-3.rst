Python 101 Module documentation
=================================

This document captures the design of WISE Python 101 Module Coverage.

Learning Outcomes
-----------------

At the end of the module, participants will be able to

	* write, test and debug the python programs
	* use functions, and represent data using python data structures
	* read and write data from and into files using python
	* understand OOPs concepts

Schedule
--------

This is a 36-hour intensive training module. The module is categorized into twelve units; Each unit is divided into three sessions; One hour each

Coverage Overview
-----------------

* Setting up the environemt 
* Loops, Strings, Lists, Sets, Tuples, and Dictionaries
* Functional Programming
* File Handling
* Object Oreintation 
* Modules - Packages
* Python UI Libraries 
* Version Control
